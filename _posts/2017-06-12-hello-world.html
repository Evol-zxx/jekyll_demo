{{ page.title }}

  • 新建一个项目目录：mkdir jekyll_demo
  • 进入根目录：cd jekyll_demo
  • 初始化项目：git init
  • 创建一个gh-pages分支，必须在此分支上改动提交代码：git checkout --orphan gh-pages
  • 在根目录下新建一个_config.yml文件: 在任意编辑器上完成即可
  • 新建一个_layouts文件夹：mkdir _layouts
  • 进入_layouts目录，新建一个default.html文件
  • 新建一个_posts目录：mkdir _posts
  • 进入_posts目录，新建一个2017-06-12-hello-world.html文件（注意：一定要是 年-月-日-名称.html 格式）
  • 添加好内容之后执行:
    1. git add .
    2. git commit -m "fist post"
    3. git remote add origin https://github.com/Evol-zxx/jekyll_demo.git (此处路径是从github上获取得，因此在进行这一步之前 要先在github上建一个和项目名相同的仓库 jekyll_demo)
    4. git push origin gh-pages
